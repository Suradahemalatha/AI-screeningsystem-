# AI-Rankingsystem-
The AI Resume Screening and Ranking System is an intelligent web application that automates the recruitment process by analyzing resumes and comparing them with a given job description. Using Natural Language Processing (NLP) and Machine Learning, the system calculates a match score for each resume, extracts key skills, and ranks .
